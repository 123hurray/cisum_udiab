## 配置文件格式

歌手   类型   列表

## config.txt示例

```
林俊杰   1   *
金莎   1   星月神话$不可思议$空气$换季
梁静茹   2   La La La La$直觉
```

## 说明

- 类型为1表示按专辑下载
- 类型为2表示按歌曲下载
- 列表为*表示下载所有歌曲
- 歌手，类型，列表间以3个空格分割
- 列表项之间以$分割
